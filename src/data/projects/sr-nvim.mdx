# **sr.nvim**

## **Overview**

sr.nvim is a Telescope-powered multi-file search and replace plugin for Neovim. It extends Neovim's capabilities by providing an efficient interface for performing search and replace operations across multiple files, leveraging the powerful Telescope framework for an enhanced user experience.

## **Features**

**Telescope Integration** – Seamless Telescope framework integration  
**Multi-File Operations** – Search and replace across multiple files  
**Neovim Plugin** – Native Neovim functionality  
**Lua Implementation** – Modern Lua-based plugin architecture  
**Enhanced Workflow** – Improved developer productivity

## **Tech Stack**

- **Editor:** Neovim
- **Language:** Lua
- **Framework:** Telescope
